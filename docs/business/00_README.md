# 00 — 全系列導讀：POS Easy Indonesia 創業手冊

> 這份文件給誰看：第一次拿到這套手冊的你。你可能完全沒有創業經驗、沒做過行銷、也不懂技術——沒關係，整套手冊就是為這樣的你寫的。
> 讀完你能做到：知道這 11 份文件各自解決什麼問題、該按什麼順序讀，並且看懂「從今天到第 6 個月」的完整創業地圖。
> 前置文件：無。這是全系列的起點。

---

## 1. 你要做的生意，一句話說完

把 **POS Easy** 這套收銀 App，賣給印尼的小店老闆——他們用了以後不會忘記誰欠錢、知道今天賺多少、手機摔壞帳也不會不見；你則收訂閱費（每月小額）或買斷費（一次付清）。

- **POS**：Point of Sale 的縮寫，就是「收銀系統」——結帳、記帳、管庫存的軟體。
- **Warung**：印尼路邊常見的家庭式雜貨小吃店，是我們的主力客群。
- **Rumah Makan**：印尼的小餐館，比 warung 大一點，是我們的第二客群。

完整的商業脈絡（為什麼賣訂閱、印尼市場怎麼打）寫在 `docs/GO_TO_MARKET.md`，那是「作戰總綱」；這套手冊則是把總綱拆成**照著做就能完成**的操作步驟。

## 2. 常用名詞小辭典（先看 10 個，之後隨用隨查）

| 名詞 | 白話解釋 |
|---|---|
| WA / WhatsApp | 印尼人最常用的通訊軟體（等於台灣的 LINE）。在印尼，生意是在 WA 裡談成的 |
| WhatsApp Business | WhatsApp 的商家版，可以放營業資訊、設自動回覆，免費 |
| QRIS | 印尼全國統一的掃碼付款系統（唸「克里斯」），店家貼一張 QR code 就能收各家電子錢包的錢 |
| Kasbon | 印尼小店的「賒帳」文化——客人先拿貨、月底結。數位化 Kasbon 是本產品規劃中的殺手級功能（開發中，對外一律說「即將推出」，口徑見 `03` §16／`06` §0） |
| Grosir | 批發商。warung 老闆每週去補貨的地方，是重要的線下推廣據點 |
| UMKM | 印尼文「中小微型企業」的縮寫，FB 上有大量 UMKM 社團，是我們找客戶的魚池 |
| 訂閱制 | 客戶每月（或每年）付固定費用，像繳電話費；不付就降回免費版 |
| 買斷制 | 客戶一次付清，軟體永久歸他，之後不用再付錢 |
| Supabase | 我們存客戶資料與授權開關的雲端後台。你在這裡「按一下」就能幫客戶開通付費功能 |
| Vercel | 免費的網站架設服務，我們的官網（Landing Page）放在上面 |

金額寫法提醒：印尼盾（Rp）用「.」當千分位。**Rp 49.000 = 四萬九千印尼盾**，約新台幣 95 元（匯率會浮動，本系列一律以 1 元新台幣 ≈ Rp 500 粗估）。印尼人口語常說 **juta**（百萬），Rp 1 juta = Rp 1.000.000 ≈ 新台幣 2 千元。

## 4. 一頁式創業全景圖：從今天到第 6 個月

整個入門期分四段。每一段都有**唯一目標**——別貪心，一段只做一件事。

```
Day 0        Week 1–4       Month 2–3        Month 4–6
 │              │               │                │
 準備 ────► Stage 0 驗證 ──► Stage 1 首批付費 ──► Stage 2 可複製通路
 開帳號      10 家白老鼠      5–10 家付費        找出 1 條可重複的獲客路
```

### Day 0｜準備（1–2 天）
- **做什麼**：註冊所有帳號、想好收款方式、建品牌資料卡。
- **用哪份文件**：01（先選模式）→ 02（照清單做）。
- **完成判準**：02 結尾的檢查清單全部打勾。

### Week 1–4｜Stage 0 驗證（目標：10 家免費白老鼠店）
- **做什麼**：在 FB 的 UMKM／warung 社團發「徵免費試用夥伴」文，一對一教學裝機，換取使用回饋。全程免費、全程手動。
- **用哪份文件**：05（逐週行動）＋ 03（怎麼介紹產品）＋ 06（線上發文方法）。
- **成功判準**：10 家中 **至少 7 家一週後還每天打開 App**。
- **⚠️ 退出條件**：若做不到 7 家留存，**停下來修產品**，不要往前衝。

### Month 2–3｜Stage 1 首批付費（目標：30–50 家安裝、5–10 家付費）
- **做什麼**：14 天全功能試用開跑；手動收款開通；從白老鼠中找出 1–2 位最熱情的老闆，養成在地代理／代收節點。
- **用哪份文件**：04（收款 SOP）＋ 07（線下與代理）＋ 08（服務與續約）。
- **成功判準**：**有人主動問怎麼付費**。

### Month 4–6｜Stage 2 可複製通路（目標：1 條可重複的獲客路徑）
- **做什麼**：把前面驗證有效的那條路（某類 FB 貼文？某個 grosir？某位代理？）加倍投入；評估啟動區域代理制（見 01 的版本 D）。
- **用哪份文件**：09（指標與覆盤）＋ 01 §版本 D。
- **成功判準**：**通路獲客成本 < 客戶 3 個月訂閱費**。

北極星指標（貫穿全程）：**每週 10 場真實對話**（WA 或面對面），勝過一切流量數據。

## 5. 全系列共同約定（每份文件都遵守）

### 5.1 方案名稱與定價（全系列唯一標準，不得出現第二套數字）

| 方案 | 價格 | 內容摘要 |
|---|---|---|
| **Gratis**（免費） | Rp 0 | 收銀＋基本庫存＋本地備份，單機完整能做生意 |
| **Paket Warung** | Rp 49.000／月 | ＋雲端備份、完整報表、進貨管理、促銷、盤點、報廢、移除收據頁尾 |
| **Paket Rumah Makan** | Rp 129.000／月 | ＋QR 點餐、會員／Kasbon、多員工＋交班＋會計 |

- 年繳 = **10 個月價**（送 2 個月）：Warung Rp 490.000／年、Rumah Makan Rp 1.290.000／年。
- **14 天全功能試用**，到期自動降級為 Gratis——資料全保留，只是付費功能收合。
- 加購裝置：Rp 20.000／台／月。
- 買斷版與早鳥終身版的價格，統一以 `01_商業模式選擇.md` 與 `04_定價與收款操作手冊.md` 內的表為準。
- **Kasbon 口徑**：Kasbon 為「開發中／即將推出」功能，全系列對外不得作現在式宣稱（口徑基準：`03` §16 與 `06` §0）。

### 5.2 一條鐵律

**免費版必須「單機完整能做生意」。永遠不鎖「今天收銀」**——客戶欠費一律降級回免費版，不是鎖死 App，資料永遠保留。這是道德底線，也是口碑來源。

### 5.3 語言約定

- 手冊本體用繁體中文。
- 所有要對印尼客戶「說出口／發出去」的內容，一律附**印尼文＋中文對照**，照抄即可使用。

## 6. 使用這套手冊的三個原則

1. **照著做就好**。每份文件的操作都寫成「第 1 步、第 2 步…」，先照抄，跑通了再優化。
2. **每週日 30 分鐘覆盤**。打開 09，對四個數字：新安裝／7 日留存／WA 進線數／免費→付費轉換。
3. **卡住就回來查**。名詞看不懂查本文件 §2；不知道現在該做什麼查 §4 全景圖找到自己所在的階段。
